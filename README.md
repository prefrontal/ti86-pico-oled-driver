# ti86-pico-oled-driver

Raspberry Pi Pico 2 OLED display driver for the Texas Instruments TI-86.

The TI-86 is a gem. One problem is that, over time, some units would develop dead lines on the display. This left you with a working calculator but an ugly screen. This project lets you swap in an OLED display driven by a Raspberry Pi Pico 2.

![TI-86 calculator with OLED screen](https://voxelwise.com/wp-content/uploads/2025/11/ti86-working-prototype.jpg)

### OLED Display

2.4" OLED display from Adafruit: https://www.adafruit.com/product/2719

This is an OLED display driven by a SSD1309 chip. It is the same resolution as the TI-86.
